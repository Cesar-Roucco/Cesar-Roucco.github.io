Senior Design Project


The Lock-Box is a project inspired by the need for some students to buy really expensive textbooks for a class. This project focused on designing a smart book-storing and book-sharing system that would allow multiple users to store a textbook in the box and the box would record whether a book was present at the moment or not. The user would need to register with the system and obtain their unique passcode to the box. This user can then add other users to the box in order to start sharing the books. The box uses pressure sensors to determine whether an object is placed on the shelf of the box (detecting a book) and detects whether a shelf is empty or full, updating a database and a website that is used to provide the user with system feedback. The system also features an LCD display that allows the user to see its passcode input. This project was the final project for my Integrated Systems Design class.

Semi-Autonomous Delivery Robot


This project was an extension to project from a previous semester. A couple of semesters back a couple of students and I built a robot from scratch that used a couple of sensors to follow a leader and stay within a certain distance. This time we changed it a little bit by making it a delivery robot that would go to a certain destination taking the shortest path. The shortest path is calculated using an algorithm that takes into account the destinations and each distance between all destinations. We also created a very simple website that allowed us to enter the destination we wanted our robot to go to, so that it can then be saved in a database and be accessed by the robot when it is doing a delivery. The robot accesses the information that was saved in a database, calculates the shortest path to that destination and produces a sequence of commands for the robot to take. We stored the list of destinations in a database and then recorded the ones that had been completed and the ones that still hadn't completed. Using the CC300 Wifi chip, we were able to access the database shown in this picture. The database kept being updated as new destinations were entered as well as when the robot completed a delivery.



Accelerometer Controlled LEDs


This project was my final project for my Embedded System Design class. It consists of an accelerometer that measures the tilt of the system from the x-y plane, as well as the direction its facing in order to control an array of LEDs that are positioned to represent cardinal directions using an MSP430 microcontroller. In the first picture, the system is leveled and therefore all LEDs shine at the same brightness. However, when the system starts to tilt, the LED closest to the ground exhibits a brighter intensity and the ones farther from the ground begin to decrease in intensity. In second picture, the LED closest to the ground is in the East position so it shines brighter than the others. In the last picture, we can see the system rotated once again, and now the LED that is closest to the ground is the one facing north, so it shines brighter than the others.



EKG Sensor Printed Circuit Board


This project was my final project for my Fundamentals of Electrical Engineering 3 class. It takes 3 inputs that are very low voltage signals from the body and is able to amplify that signal in order to be able to see a heart rate. We assembled the components of the printed circuit board and tested each part of the system individually in order to then test it as a whole. The final design for this project (3rd picture) was powered by an external battery and used different circuit components to amplify the signal and get rid of unwanted frequencies in order to isolate the heart signal. The board was completely designed and soldered by students. By the end of the testing stage, we were able to attach the electrodes to the body and see a heart rate in the monitor.
